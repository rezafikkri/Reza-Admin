Footer

  • Home
  • Footer

Membuat footer kamu membutuhkan tag <footer> dengan class .footer. Jika kamu menggunakan Sidebar, maka kamu harus menambahkan class .footer--ml-sidebar-width setelah class .footer, agar lebar Footer menyesuaikan dengan lebar Sidebar. Didalamnya kamu bisa menambahkan text menggunakan tag <p>, jika kamu punya lebih dari satu tag p, maka tambahkan class .mb-2 .mb-sm-0 pada setiap tag <p> kecuali tag <p> yang terakhir. Ini untuk memberi jarak diantara tag <p> ketika di tampilanan mobile. Letakkan footer setelah Main.
