Chart

  • Home
  • Chart

Membuat chart tidak jauh berbeda dengan dokumentasi Chart Js, namun kamu perlu tag <div> atau <section> dengan class .chart, didalamnya ada dua bagian yaitu Chart Header (tempat judul dari chart) dan Chart Body (tempat chart). Untuk membuat Chart Header, kamu perlu membuat element menggunakan tag <div> dengan class .chart__header, dan letakkan judul dari chart didalamnya menggunakan tag <h6>. Untuk membuat Chart Body kamu perlu membuat element menggunakan tag <div> dengan class .chart__body, didalamnya letakkan code untuk chart, dan letakkan Chart Body dibawah Chart Header. Standar tinggi dari chart adalah 350px, kamu juga bisa mengaturnya dengan menggunakan attribute height pada tag canvas.

Jika kamu belum pernah menggunakan Chart js sebelumnya, saya sarankan untuk membaca dokumentasi dari Chart js!. Disana sudah dijelaskan dengan detail. Jika kamu sebelumnya pernah mempelajari cara membuat Main pada menu Tata Letak > Main, kamu bisa meletakkan Chart didalam grid sistem Bootstrap, dan tanpa harus meletakkan Chart didalam Box.

Area Chart

Pengunjung, Januari 2020
CopyHTML
<section class="chart">
    <div class="chart__header">
        <h6>Pengunjung, Januari 2020</h6>
    </div>
    <div class="chart__body">
        <canvas id="area_chart"></canvas>
    </div>
</section>
CopyJavascript
// Area Chart
const areaChart = document.querySelector("#area_chart").getContext('2d');
let configAreaChart = {
    type: 'line',
    data: {
        labels: ['Ahad, 11','Senin, 12','Selasa, 13','Rabu, 14','Kamis, 15','Jum\'at, 16'],
        datasets: [{
            label: 'Pengunjung',
            data: [10,6,7,5,1,14],
            fill: 'origin',
            backgroundColor: 'rgba(37,151,224,.7)',
            borderColor: '#2597e0'
        }]
    },
    options: {
        maintainAspectRatio: false,
        legend: {
            display: false
        },
        tooltips: {
            titleFontFamily: 'sans-serif',
            bodyFontFamily: 'sans-serif',
            backgroundColor: '#fff',
            titleFontColor: '#333',
            bodyFontColor: '#333',
            borderColor: '#e2e2e2',
            borderWidth: '1',
        }
    }
}
new Chart(areaChart, configAreaChart);
